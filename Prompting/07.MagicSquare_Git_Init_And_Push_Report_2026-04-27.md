## 1. Epic

  - **Concept:** 부분 보드(0 정확히 2칸)를 유효성 검사하고, 도메인 규칙으로 해를 구하거나 표준 오류로 거부한다.

## 2. User Story

- [ ] **US-001 (Track A / FR-01)** — *입력이 계약을 만족할 때만* 도메인으로 넘기고, 위반 시 **표준 오류 코드**로 거부한다. (AC-FR01-01~05, TC-04~07)
