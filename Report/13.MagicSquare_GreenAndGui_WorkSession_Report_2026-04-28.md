## 3. GUI 실행 경로 추가 (Screen 레이어)

### 3.1 추가 파일

| 파일 | 역할 |
|------|------|
| [`../src/magicsquare/gui/__init__.py`](../src/magicsquare/gui/__init__.py) | GUI 패키지 선언 |
| [`../src/magicsquare/gui/__main__.py`](../src/magicsquare/gui/__main__.py) | 공식 실행 진입점 (`python -m magicsquare.gui`) |
| [`../src/magicsquare/gui/app.py`](../src/magicsquare/gui/app.py) | 4x4 입력 UI, Solve 버튼, 결과/오류 표시 |

### 3.2 의존성

- [`../pyproject.toml`](../pyproject.toml) `optional-dependencies.gui`에 `PyQt6>=6.7` 추가
- 설치 명령: `pip install -e ".[gui]"`

### 3.3 MVP 동작

- 4x4 입력 필드(`QLineEdit`)
- `Solve` 버튼 클릭 시:
  1) 입력 파싱  
  2) `boundary.validate()`  
  3) `domain.find_blank_coords()`  
  4) 결과 라벨 업데이트(현재는 preview 형태)
- 오류 시 `QMessageBox` 표시
